# Unanimy

## Local development

1. Install dependencies:
   ```bash
   cd web
   npm install
   ```
2. Copy env vars and fill in values:
   ```bash
   cp .env.example .env.local
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Run checks:
   ```bash
   npm run lint
   npm run build
   ```
